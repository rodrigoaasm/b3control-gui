# B3control

This repository is a gui for b3 control (https://github.com/rodrigoaasm/b3control). 

## Quick start

- Install dependencies: `npm install` or `yarn`

- Start the server: `npm run dev` or `yarn dev`

- Views are on: `localhost:3000`
